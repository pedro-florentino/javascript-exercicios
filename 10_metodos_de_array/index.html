Métodos de Array em JavaScript

Aprenda a usar 10 métodos de array aplicados no código!

De uma olhada no console da página também.

Data:

Os dados da explicação.

const data = [
{ name: "Matheus", age: 31, sallary: 2000, driverLicense: true },
{ name: "João", age: 18, sallary: 1500, driverLicense: false },
{ name: "Mariana", age: 22, sallary: 4000, driverLicense: true },
{ name: "Pedro", age: 50, sallary: 7200, driverLicense: true },
{ name: "Érica", age: 16, sallary: 0, driverLicense: false },
];

1. Reverse

O método reverse() inverte a ordem dos elementos do array original.

const reverseData = data.reverse();
console.log(reverseData);

Resultado:

[
{ "name": "Érica", "age": 16, "sallary": 0, "driverLicense": false },
{ "name": "Pedro", "age": 50, "sallary": 7200, "driverLicense": true },
{ "name": "Mariana", "age": 22, "sallary": 4000, "driverLicense": true, },
{ "name": "João", "age": 18, "sallary": 1500, "driverLicense": false },
{ "name": "Matheus", "age": 31, "sallary": 2000, "driverLicense": true }
]

2. Find

O método find() retorna o primeiro elemento que atende a uma condição específica.

const highestSallary = data.find(user => user.sallary > 5000);
console.log(highestSallary);

Resultado:

{ "name": "Pedro", "age": 50, "sallary": 7200, "driverLicense": true }

3. FindIndex

O método findIndex() retorna o índice do primeiro elemento que atende a uma condição.

const lowestSallary = data.findIndex(user => user.sallary > 0 && user.sallary < 2000);
data[lowestSallary].sallary += 200;
console.log(data);

Resultado:

3
{ "name": "João", "age": 18, "sallary": 1700, "driverLicense": false }

4. Includes

O método includes() verifica se um array contém um elemento específico e retorna true ou false.

const numbers = [1, 2, 3, 4, 5];
const hasFour = numbers.includes(4);
console.log(hasFour);

Resultado:

true

5. Map

O método map() cria um novo array modificando cada elemento do array original.

data.map(user => user.newsletter = false);
console.log(data);

Resultado:

[
{ "name": "Érica", "age": 16, "sallary": 0, "driverLicense": false, "newsletter": true },
{ "name": "Pedro", "age": 50, "sallary": 7200, "driverLicense": true, "newsletter": false },
{ "name": "Mariana", "age": 22, "sallary": 4000, "driverLicense": true, "newsletter": false },
{ "name": "João", "age": 18, "sallary": 1700, "driverLicense": false, "newsletter": false },
{ "name": "Matheus", "age": 31, "sallary": 2000, "driverLicense": true, "newsletter": false }
]

6. Filter

O método filter() cria um novo array contendo apenas os elementos que atendem a uma condição.

const drivers = data.filter(user => user.driverLicense);
console.log(drivers);

Resultado:

[
{ "name": "Pedro", "age": 50, "sallary": 7200, "driverLicense": true, "newsletter": false },
{ "name": "Mariana", "age": 22, "sallary": 4000, "driverLicense": true, "newsletter": false },
{ "name": "Matheus", "age": 31, "sallary": 2000, "driverLicense": true, "newsletter": false }
]

7. Reduce

O método reduce() reduz o array a um único valor, aplicando uma função acumuladora.

const sallariesSum = data.reduce((totalSallary, user) => totalSallary += user.sallary, 0);
console.log(sallariesSum);

Resultado:

14900

8. ForEach

O método forEach() executa uma função para cada elemento do array.

data.forEach(user => {
console.log(`Olá ${user.name}!`);
});

Resultado:

Olá Érica!
Olá Pedro!
Olá Mariana!
Olá João!
Olá Matheus!

9. Some

O método some() verifica se pelo menos um elemento do array atende a uma condição.

let someoneWithNewsletter = data.some(user => user.newsletter);
console.log(someoneWithNewsletter);

Resultado:

true

10. Every

O método every() verifica se todos os elementos do array atendem a uma condição.

const everyUserHasName = data.every(user => user.name);
const everyUserHasGoodSallary = data.every(user => user.sallary > 2000);
console.log(everyUserHasName);
console.log(everyUserHasGoodSallary);

Resultado:

true
false
